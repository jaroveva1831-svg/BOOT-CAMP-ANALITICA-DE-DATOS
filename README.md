# 📊 Análisis de Datos con Python y Google Colab

Proyecto de estudio orientado al aprendizaje práctico de análisis exploratorio de datos (EDA), limpieza de datos, visualización y técnicas básicas de ciencia de datos usando Python en Google Colab. [web:1][web:4]

## 🎯 Objetivo del proyecto

Este repositorio tiene como objetivo:
- Practicar el flujo completo de análisis de datos: carga, limpieza, exploración y visualización.
- Consolidar el manejo de librerías clave de ciencia de datos en Python.
- Mostrar un trabajo profesional y bien documentado, útil como portafolio de estudio.

## 🧰 Tecnologías y librerías

El proyecto está desarrollado en **Python** utilizando **Google Colab** como entorno principal. [web:1][web:6]  
Las principales librerías utilizadas son:

- `pandas` – manipulación y análisis de datos.
- `numpy` – operaciones numéricas.
- `matplotlib` y/o `seaborn` – visualización de datos.
- `plotly` (opcional) – gráficos interactivos.

## 📂 Estructura del repositorio

- `notebooks/` – carpeta que contiene los notebooks de Colab.
  - `01_eda.ipynb` – Análisis exploratorio de datos.
  - `02_visualizacion.ipynb` – Gráficos y visualización avanzada.
- `data/` – carpeta con el conjunto de datos utilizado (CSV, Excel, etc.).
- `README.md` – este archivo de documentación.

## 🚀 Cómo usar el proyecto

1. Abre el notebook correspondiente en **Google Colab** (puedes hacer clic en el enlace de “Abrir en Colab” si lo agregas, o descargar el `.ipynb` y subirlo a Colab).
2. Ejecuta las celdas de código en orden.
3. Revisa los comentarios en cada celda para entender el propósito del análisis. [web:4][web:6]

> ⚠️ Este proyecto es de **estudio**, por lo que puedes modificarlo, experimentar con otros datasets y añadir nuevas visualizaciones o técnicas.

## 📚 Notas de estudio

- Cada notebook está pensado para seguir un flujo lógico: carga → limpieza → exploración → visualización.
- El código está comentado para facilitar el aprendizaje y la revisión futura.
- Se recomienda usar este repositorio como base para practicar nuevas técnicas (modelos básicos de regresión, clusterización, etc.) una vez dominado el análisis exploratorio.
